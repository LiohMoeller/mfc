SYSTEM REQUIREMENTS
- Integration in Ubuntu Unity, <https://wiki.ubuntu.com/Unity/LauncherAPI>
- Dynamic Icon in Unity Launcher
- Status Information, graphical, how mucht time


USER INTERFACE
- Clock with notifications in specified intervals
- Text messages
- Audio messages
- User specified interval
- User specified text messages
- User specified audio messages


PROGRAMMING LANGUAGE
- Python
- wxPython (GUI)
- PyGame (Sound)


VERSION NUMBERS
0: Development version.
1: First version with full ubuntu unity integration


OOP
UumfcGUI (wx.Frame)
 |
 |--UumfcData()
 |
 |--UumfcMsg(wx.Dialog)
 |
 |--UumfcUnity()
 

UumfcGUI
- Text entry for time interval in minutes.
- Buttons to increase or decrease the minutes.
- Audio file description.
- Buttons to change audio file and for sound preview.
- Activate or deactivate UUmfc.
- Close and Minimize UUmfc.
- Timer event for text and sound notification
- Sound with PyGame
- Load settings at startup, save settings at program end.

UumfcData
- User Specifications.
- Program Options.
- Get and set settings as text.

UumfcMsg
- Pop up message text.

UumfcUnity
- Launcher integration.
- Dynamic program icon.
- status indicator in Launcher of the remaining time.
